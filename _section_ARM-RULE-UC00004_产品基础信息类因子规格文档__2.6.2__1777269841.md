### 因子参数

无

### 因子算法说明

#### 因子取值逻辑：

1.  整体逻辑说明：

- 该因子仅统计债券、存款；【证券范围无关】

- 计算公式：组合固收资产ESG得分=![](media/image6.wmf)（债券全价市值\*发行主体中债ESG评级+存款余额\*发行主体中债ESG评级）/（债券全价市值+存款余额）

- 发行主体ESG评分为空（枚举为-2）的不进行计算；【机构基本信息表中DWD_PTY_INST_BASI_INFO的中债ESG评级CNBD_ESG_RAT为空的不纳入计算】

- 债券市值算法为全价市值：即现货持仓市值因子，其他参数默认选项，"债券风控市值计算类型"按"2-\>全价"处理；

- 存款余额计算时含息：即存款余额因子，其他参数默认选项，参数"10070
  利息计算方式"="0-\>含息"的计算方式；

2.  整体业务说明：

+----------+-----------+----------------------------------------------------------------------+--------------------------+------------------------------------------------------------------------------------------------------------------+
| 监控模式 | 业务环节  | 业务说明                                                             | 趋势判断变量             | 说明                                                                                                             |
+----------+-----------+----------------------------------------------------------------------+--------------------------+------------------------------------------------------------------------------------------------------------------+
| 事前     | 指令端    | 同通用公式，其中债券市值/存款余额取指令端的债券市值/存款余额计算逻辑 | 同现货持仓市值、存款余额 | （1）债券市值计算：同"现货持仓市值"因子：证券类别选择债券，"债券风控市值计算类型"按"2-\>全价"处理，其余参数默认 |
|          |           |                                                                      |                          |                                                                                                                  |
|          |           |                                                                      |                          | （2）存款余额计算：同"存款余额"因子，参数"10070 利息计算方式"="0-\>含息"进行计算，其他参数默认                   |
|          +-----------+----------------------------------------------------------------------+--------------------------+                                                                                                                  |
